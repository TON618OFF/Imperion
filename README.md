## О проекте

Imperion-Pro - это образовательная платформа, которая позволяет изучать различные языки программирования через интерактивные уроки и лекции. Платформа включает в себя:

- Практические уроки с проверкой кода
- Теоретические лекции
- Систему достижений и опыта (XP)
- Отслеживание прогресса обучения
- Панель администратора для управления
- Панель ментора для создания контента

## Поддерживаемые языки программирования

- Python
- JavaScript
- C++
- C
- Go

## Технологический стек

- **Frontend**: React + TypeScript
- **UI**: Radix UI + Tailwind CSS
- **Backend**: Supabase (PostgreSQL + Auth + Storage)
- **Code Execution**: Piston API
- **Build Tool**: Vite

## Установка

### Требования

- Node.js (версия 16 или выше)
- npm или yarn

### Установка зависимостей

```bash
npm install
```

## Разработка

Запуск локального сервера разработки:

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка

Для создания production сборки:

```bash
npm run build
```

Для предпросмотра production сборки:

```bash
npm run preview
```

## Тестирование

Запуск тестов:

```bash
npm test
```

Запуск тестов в режиме watch:

```bash
npm run test:watch
```

## Структура проекта

```
Imperion-Pro/
├── src/
│   ├── auth/          # Аутентификация
│   ├── components/    # React компоненты
│   ├── hooks/         # Custom React hooks
│   ├── lib/           # Утилиты и вспомогательные функции
│   ├── pages/         # Страницы приложения
│   └── test/          # Тесты
├── supabase/
```

## Конфигурация базы данных

Схема базы данных находится в файле `supabase/schema.sql`. Для применения изменений в базе данных Supabase, выполните SQL из этого файла в SQL Editor вашего проекта Supabase.

📖 **Подробная инструкция:** См. [DATABASE_SETUP.md](DATABASE_SETUP.md)

## Роли пользователей

### 👤 Пользователь
- Доступ ко всем урокам и лекциям
- Отслеживание прогресса обучения
- Получение достижений и XP
- Редактирование своего профиля

### 👨‍🏫 Ментор
- Все права пользователя
- Доступ к **Панели ментора** (`/mentor`)
- Создание, редактирование и удаление уроков
- Создание, редактирование и удаление лекций
- Управление контентом для всех языков

### 🛡️ Администратор
- Все права ментора
- Доступ к **Панели администратора** (`/admin`)
- Просмотр статистики и аналитики
- Управление пользователями
- Экспорт данных в PDF

## Переменные окружения

Создайте файл `.env.local` в корне проекта:

```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```
